An analogy to describe JavaScript and its relationship to HTML and CSS

The relationship between JavaScript and HTML/CSS is an interesting reslationship to witness and write about. To describe this relationship, i think it would be best to imagine working as a painter on a canvas.

HTML is the basic sketch on the canvas. Its the essential structure of the painting, this is where we place objects and elements. Without this intial sketch, there would be no way of guiding trhe structing to the page.

CSS is the fun part, we can imagine it as the all the colors and brush strokes a painter would use to make the painting awesome and nice to look at. The same thing applies to the web page. With CSS we can add a creative flare to our elements on the page. Like for example changing the text to green .

Finally JavaScript could be considered the interactive part of the painting if there was one. Imagine being able to make the painted stars move on Van Gogh 'The Starry Night'. In otherwords JavaScript makes the elements more dynamic and interactive, and can change overtime.

Describe what the DOM is and an example of how you might interact with it.

The DOM is a abbreviation for the Document Object Model. The DOM bassically connects web pages to the different scripts and languages that we use, eg HTML. this gives it structure. This structure we can call the DOM tree. The DOM tree represents all the different nodes and elements that make up the structure of a page. Javascript has the power to change any of the properties of these nodes by either adding to them, or you can even remove some of the elements, pretty cool if you ask me.
